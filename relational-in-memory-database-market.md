<p><ul><li>Main Memory Database (MMDB)</li><li>Real-time Database (RTDB)</li></ul></p>
<p><p>Relational In-Memory Database market comprises two major types, namely Main Memory Database (MMDB) and Real-time Database (RTDB). MMDBs store data entirely in the computer's main memory, providing faster access and improved performance compared to traditional disk-based databases. They are commonly used in applications requiring real-time analysis and processing of large data sets. On the other hand, RTDBs handle high-velocity data streams and prioritize capturing and processing real-time data, making them ideal for applications that demand instant data availability and real-time decision-making capabilities.</p></p>
